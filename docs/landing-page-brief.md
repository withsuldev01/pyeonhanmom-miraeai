# 편한몸 의원 부설 미래아이 심리발달 클리닉 랜딩 페이지 기획 초안

## 1. 목적

아동 발달이 걱정되는 보호자가 센터를 빠르게 이해하고, 초기 상담 또는 전화 상담으로 자연스럽게 전환하도록 돕는 홍보형 랜딩 페이지를 만든다.

핵심 전환은 다음 두 가지다.

- `상담 문의` / `상담 신청`: 외부 아동 발달 초기 상담 평가지로 이동
- `전화상담`: 전화 연결 또는 전화번호 안내

## 2. 현재 제공 자료

- `docs/편한몸홈페이지.pdf`: 비전문가가 작성한 와이어프레임 및 섹션 구성안
- `docs/logo.png`: 센터 로고
- `docs/center-lobby.jpeg`: 센터 로비 실사 이미지

PDF는 총 8쪽이며, 랜딩 페이지의 대략적인 섹션 흐름과 주요 카피가 포함되어 있다. 로고는 가로형 락업이며, 실제 표기는 `편한몸 의원 부설 미래아이 심리발달 클리닉`으로 확인된다.

## 3. 페이지 톤

- 보호자가 불안을 느끼는 상황에서 부담 없이 상담을 시작할 수 있는 안정감
- 의료/발달 전문성은 유지하되, 차갑거나 과장된 표현은 피함
- 아이와 보호자를 함께 바라보는 다정한 문장 톤
- 센터 실사 이미지와 로고 색상을 활용해 신뢰감과 실제감을 강화

## 4. 정보 구조

### 4.1 상단 네비게이션

- 좌측: 로고
- 우측 주요 액션: `전화상담`
- 보조 메뉴는 필요 시 `프로그램`, `검사 과정`, `후기`, `상담 신청` 정도로 제한

### 4.2 히어로

목표: 첫 화면에서 어떤 센터인지, 어떤 도움을 받을 수 있는지, 바로 상담할 수 있는지를 전달한다.

주요 카피 후보:

```text
아이의 발달을
더 넓고 깊게 살핍니다
```

보조 카피:

```text
아이의 발달은 한 가지 영역만으로 설명되지 않습니다.
언어, 인지, 운동, 감각, 사회성 등 전반적인 발달 상태를 체계적으로 평가하고
아이에게 필요한 맞춤 치료교육을 통해 건강한 성장을 함께 만들어갑니다.
```

주요 CTA:

- `상담 문의`
- `전화상담`

이미지:

- PDF의 참고 이미지 대신 현재 저장된 `center-lobby.jpeg`를 우선 활용
- 실제 센터 공간이 보이도록 과도한 블러, 어두운 오버레이, 장식성 크롭은 피함

### 4.3 상담 평가지 안내

목표: CTA를 누르면 이동하는 외부 초기 상담 평가지의 성격을 미리 설명한다.

연결 URL:

```text
https://child-developmental-assessment-form-sage.vercel.app/
```

PDF 2쪽과 마지막 쪽에 있는 화면은 위 URL의 초기 화면 참고 이미지다. 랜딩 페이지 안에 동일한 폼을 직접 구현하는 것이 아니라, `상담 문의` 또는 `상담 신청` 버튼을 클릭했을 때 해당 외부 페이지로 이동시키는 방식으로 본다.

내용:

- 제목: `아동 발달 초기 상담 평가지`
- 안내: 상담 전 간단한 문진을 작성하며 약 5분 정도 소요
- 개인정보 수집/이용 및 결과 수신 동의 필요
- 시작 버튼 제공
- 랜딩 페이지 하단에서는 상담 평가지 화면을 직접 배치하기보다, 상담 신청 CTA와 외부 이동 안내를 배치

### 4.4 발달 지원 방향

목표: 센터가 단순 치료 나열이 아니라 평가, 목표 설정, 일상 연계까지 함께 본다는 메시지를 전달한다.

카드 3개:

1. 현재 상태를 정확히 이해합니다.
   - 언어, 인지, 감각, 운동, 사회성 발달을 종합적으로 살펴 어떤 도움이 필요한지 확인
2. 아이에게 맞는 목표를 세웁니다.
   - 발달 수준과 생활 속 어려움에 맞춰 무리하지 않고 이어갈 수 있는 치료교육 방향 계획
3. 일상으로 이어지는 변화를 돕습니다.
   - 보호자와 변화를 공유하고 가정과 기관 생활에서도 성장 경험이 이어지도록 지원

### 4.5 프로그램

목표: 제공 치료 영역을 한눈에 보여준다.

치료 프로그램:

- 언어치료
- 감각통합치료
- 인지치료
- 놀이치료
- 미술치료
- 짝치료
- 그룹치료

구현 방향:

- 2열 또는 3열 카드형 구성
- 프로그램별 간단한 아이콘 또는 일러스트 사용
- 모바일에서는 1열 카드로 자연스럽게 전환

### 4.6 치료사 소개

목표: 아이와 보호자를 만나는 사람들에 대한 친근감과 신뢰를 만든다.

PDF 방향:

- 가로로 넘기며 확인 가능한 치료사 카드
- 현재 PDF에는 실제 인물 정보 대신 캐릭터형 예시가 들어 있음

문서 단계 결정 필요:

- 실제 치료사 이름/전문분야/자격/사진을 노출할지
- 캐릭터형 익명 소개로 갈지
- 원장님 소개를 별도 강조할지

### 4.7 검사 및 치료 과정

목표: 상담 이후 어떤 절차로 진행되는지 보호자가 예측할 수 있게 한다.

단계:

1. 원장님 초진 및 발달 확인
2. 발달 전문가 심층 검사
3. 검사 결과 해석상담
4. 1회 치료 경험 및 향후 진행 방향 확인

이 섹션은 타임라인 또는 단계형 레이아웃이 적합하다.

### 4.8 초기 발달 평가 패키지

목표: 상담을 망설이는 보호자에게 시작 비용과 구성 정보를 명확히 제시한다.

카피:

```text
편한몸, 편한마음으로 시작하는 발달치료
```

구성:

```text
원장님 초진 + 보호자 체크리스트 + 심층 검사 + 해석상담 + 1회 치료
총 60,000원
```

주의:

- 가격과 구성은 최종 확인 필요
- 의료/상담 서비스 표현이므로 과장된 성과 보장 문구는 피함

### 4.9 보호자 후기

목표: 실제 변화 사례와 보호자 안심 포인트를 전달한다.

PDF에 포함된 후기 문구는 10개다. 실제 페이지에서는 전체를 표로 보여주기보다 카드형 슬라이더 또는 3-6개 대표 후기 노출이 적합하다.

후기 예시:

- 처음엔 낯설어하던 아이가 치료 시간을 기다려요.
- 말로 표현하는 일이 조금씩 늘고 있어요.
- 초진 후 상담까지 이어져 안심됐어요.
- 막막했던 치료 방향이 잡혔어요.
- 치료 후 피드백이 자세해서 좋아요.

확인 필요:

- 실제 보호자 후기인지, 예시 카피인지
- 별점/보호자 표기/익명 처리 방식
- 의료 광고 관련 표현 검토 필요 여부

### 4.10 마지막 CTA

목표: 페이지를 읽은 보호자가 상담 신청으로 이동하도록 마무리한다.

주요 카피:

```text
아이의 작은 신호에도,
함께 마음을 기울입니다.
```

보조 카피:

```text
원장님과 다양한 치료사들이 함께 아이를 바라보며
보호자님의 마음까지 다정하게 헤아립니다.
```

CTA:

- `상담신청`
- 외부 초기 상담 평가지로 이동

## 5. 전환 동선

- 히어로 `상담 문의` 버튼 -> `https://child-developmental-assessment-form-sage.vercel.app/`
- 상단 `전화상담` 버튼 -> 전화 연결 또는 전화번호 안내
- 하단 `상담신청` 버튼 -> `https://child-developmental-assessment-form-sage.vercel.app/`

상담 평가지 동선은 외부 URL 이동으로 확정한다.

구현 시 선택할 세부 방식:

- 현재 탭에서 이동
- 새 탭으로 열기
- 버튼 클릭 전 짧은 안내 문구 표시 후 이동

## 6. 디자인 방향

- 로고의 네이비, 청록, 노랑, 빨강, 초록 계열을 포인트 컬러로 사용
- 배경은 흰색 또는 아주 옅은 웜그레이 중심
- 병원/센터의 신뢰감과 아동 발달 센터의 부드러움을 함께 반영
- 과도한 캐릭터 장식보다 실제 공간 사진, 명확한 정보, 적절한 아이콘 중심
- 카드 모서리는 과하게 둥글지 않게 8px 내외로 제한
- 모바일에서 CTA가 잘 보이도록 하단 고정 상담 버튼 검토

## 7. 미확정 항목

- 최종 브랜드 표기: `편한몸 의원 부설 미래아이 심리발달 클리닉`으로 통일할지
- PDF 안의 `코아` 문구를 모두 `미래아이` 또는 센터명으로 교체할지
- 전화상담 연결 전화번호
- 치료사 소개에 실제 인물 정보 사용 여부
- 프로그램별 실제 제공 여부와 명칭
- 패키지 구성과 가격 `60,000원` 확정 여부
- 후기 문구의 실제 사용 가능 여부
- 사업장 주소, 진료/상담 시간, 지도 노출 여부

3. 치료 프로그램/패키지/후기 사용 가능 여부 확인
